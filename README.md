### Introduction
The aim of this project, regardless of the speculative bubbles in Bitcoin price, is to give a hint on the future closing price values. 


### Environment Setup
To run the Prediction Models the following should be installed:

1. Python 3+
2. Tensorflow=1.10.1
3. Keras=2.2.2
4. Pandas=0.23.4
5. Numpy=1.15.1
6. Matplotlib=2.2.3
7. sklearn=0.19.2
